# Stack-Overflow-Developer-Survey-Analysis
Статистичний аналіз профілю IT-спеціаліста за допомогою Python (Pandas)

## Опис проєкту
Дослідження масштабного датасету опитування Stack Overflow (понад 60,000 респондентів). Мета — вивчити зв'язок між рівнем компенсації, досвідом роботи та форматом зайнятості (remote/office) у різних країнах.

## Інструменти та технології
Python: основна мова обробки даних.
Pandas & NumPy: очищення даних, багатофакторна фільтрація та агрегація.
Jupyter Notebook: середовище розробки.

## Що було зроблено (Технічна частина):
Data Cleaning & Preprocessing: виявлення та обробка пропущених значень у ключових атрибутах (Salary, YearsCode, WorkLoc).
Сегментація даних: розріз аудиторії за типом зайнятості, форматом роботи (Remote/Office) та географією.
Статистичний аналіз: розрахунок мір центральної тенденції (середнє, медіана, мода) для оцінки розподілу доходів.
Агрегування: порівняльне групування показників компенсації в розрізі досвіду роботи та формату зайнятості.

## 💡 Ключовий висновок:
Проведено комплексний аналіз факторів, що корелюють із рівнем компенсації спеціалістів. Виявлено відмінності в розподілі доходів залежно від формату роботи (Remote vs Office) та рівня досвіду. Детальні цифрові показники та залежності зафіксовані в розрахунках усередині Jupyter Notebook.
